这个教程将会指引你与 Review.md 分享创意

在网站的首页,你可以看到一个妙趣横生的标语.而它是由用户自行创作的.

  • 点击这个链接,创建一个你自己的 Github 账号.
    • -> GITHUB
  • 访问位于 Github 的 Review.md 的创意贡献地址, 点击右侧的 fork.
    • -> Review.md 在 Github 的地址
  • 在跳转的界面里,点击文件 Review.title 点击文本查看器右上角的编辑.
  • 完成对文件的编辑之后保存, 点击发送 Pull Request, 仅两三分钟即可完成.

    我们强烈建议你尝试并使用 Github, 但你也可以发送邮件到 llf@wmpcxpy.com 提交你的作品.

    同样的, 你也可以发送邮件到上述邮箱请求公开你创建的题库.
